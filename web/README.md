# docpull web

Marketing website for [docpull](https://github.com/raintree-technology/docpull) - the public web-source fetcher for AI agents.

## Tech Stack

- Next.js 16
- React 19
- TypeScript
- Tailwind CSS
- Framer Motion

## Development

```bash
npm install
npm run dev
```

Open [http://localhost:3013](http://localhost:3013).

## Build

```bash
npm run build
npm run start
```
