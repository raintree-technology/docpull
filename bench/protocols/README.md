# Frozen benchmark protocols

Files in this directory predeclare future evaluation configurations. Freezing a
protocol does not authorize network requests, provider spend, or a comparative
claim.

`future-extraction-quality-v1.yaml` records the quality-max extraction profiles
for DocPull 6.1.0/pypdf, Exa Full, Parallel Full, Tavily Advanced, and
Firecrawl. It is reserved for the external sealed-holdout process and has not
been executed by the repository-hosted workflow.
